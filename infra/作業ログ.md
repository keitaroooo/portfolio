## Cloudflare Pages

コード: [`infra/terraform/cloudflare/`](terraform/cloudflare/)

| リソース | 説明 |
| --- | --- |
| `cloudflare_pages_project.portfolio` | Cloudflare Pages プロジェクト（静的サイト, root: `src`） |
| `cloudflare_pages_domain.www` | カスタムドメイン `www.keitaroooo.com` |

### 環境変数の準備（fish の例）

```fish
function load_cloudflare_tokens
    set -gx BW_SESSION (bw unlock --raw)
    set -gx TF_VAR_cloudflare_api_token (bw get password "Cloudflare API Token")
    set -gx TF_VAR_cloudflare_account_id (bw get password "Cloudflare Account ID")
    set -gx TF_VAR_zone_id (bw get password "Cloudflare Zone ID")
end
```

### 初回セットアップ

```bash
cd infra/terraform/cloudflare
load_cloudflare_tokens
terraform init
terraform plan
terraform apply
```

### 既存リソースの import（必要な場合）

```bash
# Cloudflare ダッシュボードからプロジェクト名を確認
terraform import cloudflare_pages_project.portfolio portfolio
terraform import cloudflare_pages_domain.www portfolio/www.keitaroooo.com
```
